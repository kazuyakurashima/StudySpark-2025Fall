## 作成後の確認

以下のSQLで作成されたユーザーのUUIDを確認してください：

```sql
SELECT id, email, raw_user_meta_data->>'role' as role, raw_user_meta_data->>'full_name' as full_name
FROM auth.users
ORDER BY email;
```

これらのUUIDをメモして、STEP 3で使用します。
